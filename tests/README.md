# Тесты для MovieHub Backend

Простые CRUD тесты для основных операций API.

## Запуск тестов

```bash
# Установить зависимости
pip install -r requirements.txt

# Запустить все тесты
pytest

# Запустить с подробным выводом
pytest -v

# Запустить конкретный файл тестов
pytest tests/test_users.py
```

## Структура тестов

- `test_users.py` - Тесты для регистрации, входа и получения пользователя
- `test_movies.py` - Тесты для получения списка фильмов и поиска
- `test_reviews.py` - Тесты для создания, обновления и удаления отзывов
- `test_lists.py` - Тесты для создания, обновления и удаления списков фильмов

## Примечания

- Тесты используют SQLite в памяти для изоляции
- Каждый тест создает свою БД и удаляет её после выполнения
- Для тестов, требующих авторизации, используется фикстура `auth_token`
